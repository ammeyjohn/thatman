# ThatMan-文字修仙游戏

本作是一款沉浸式单人文字修仙游戏，以真实时辰流逝、自由剧情抉择、慢节奏修仙养成为核心。玩家将从普通修士起步，拜入宗门，经历拜师、修行、实战、突破境界等修仙日常。
本作整体操作简单易上手，依托文字交互与选择驱动剧情，主打自由养成与沉浸式修仙体验。同时具备两大核心特点：
- 高自由养成：无强制绑定玩法，玩家可自由安排修行节奏，自主进行闭关修炼、地图探索、处理宗门事务、搜集稀缺资源，按照自身喜好塑造专属修士成长路线。
- AI动态内容：摒弃固定脚本，由AI全权生成全部游戏内容，包含大世界场景、NPC人设与对话、支线剧情、随机奇遇、突发事件；内容动态适配玩家境界、过往决策与行为习惯，规避流水线剧情，每次游玩都拥有独一无二的修仙历程。

## 文件系统
- game_manual.md
阐述游戏定位、核心特色、运行规则与交互指令，面向玩家，作为游戏入门指导文档。
- world_config.md
统筹地图、天气两类环境数据，设定环境增益与负面效果，影响修炼、资源、奇遇，辅助AI生成动态场景。
- task_config.md
规范任务难度分级、类型及奖励区间，为AI生成各类任务提供标准，平衡任务收益与角色境界。
- level_config.md
定义完整修士境界体系，制定突破、属性、权限相关规则，是角色修炼、战力计算的底层数据基础。
- item_config.md
统一管理全品类道具资源，规范道具属性、获取方式与使用效果，支撑AI掉落、兑换、奖励等功能。
- skill_config.md
制定各类功法品质与属性规则，明确习得及升级条件，为AI生成功法、更新宗门功法库提供依据。
- npc_config.md
划分NPC类型与行为逻辑，规范AI生成NPC的各项参数，适配交易、奇遇、战斗等全部交互场景。
- guild_config.md
设定宗门势力分级标准，管控宗门资源、弟子体系与势力关系，辅助AI生成宗门及处理宗门相关玩法。
