# 页面设计文档

## 技术体系
- React+Vite+TypeScript
- Tailwind CSS（任意值语法）
- Zustand 状态管理

## 设计概述

### 设计理念
以"上古劫后遗落世道"为核心，营造深邃、神秘的修仙氛围。界面采用深色系为主，配合古朴的文字排版，让玩家仿佛置身于残破的修仙典籍之中。

### 核心氛围词
- **古朴**：如上古残卷，承载岁月痕迹
- **深邃**：如青墟之夜，蕴含无尽可能
- **静谧**：如闭关洞府，修心之所
- **玄妙**：如天地法则，因果循环

## 布局结构

### 三栏布局
```
┌──────────┬─────────────────────────┬──────────┐
│          │                         │          │
│  左侧面板  │      中间聊天区域          │  右侧面板  │
│  280px   │      flex-1             │  280px   │
│          │                         │          │
│ 角色状态   │      ChatArea           │ 世界事件   │
│          │                         │          │
└──────────┴─────────────────────────┴──────────┘
```

- 左右面板可折叠，折叠时宽度为 0，通过按钮切换
- 折叠/展开按钮位于面板与中间区域的交界处，宽 24px、高 48px
- 背景色 `#0a0a0f`，面板背景 `#0d1f1f`

### 消息气泡样式
- **玩家消息**：渐变 `from-[#2d5a5a] to-[#1a3a3a]`，边框 `border-[#3d7a7a]/50`，右上角切角
- **NPC 消息**：渐变 `from-[#1a2f2f] to-[#0d1f1f]`，边框 `border-[#2d5a5a]/50`，左上角切角
- **系统消息**：居中胶囊，背景 `bg-[#2d5a5a]/20`，边框 `border-[#2d5a5a]/40`
- **事件消息**：居中金色渐变边框，`from-[#c9a227]/20 via-[#c9a227]/10 to-[#c9a227]/20`

### 头像样式
- **玩家头像**：渐变 `from-[#3d7a7a] to-[#2d5a5a]`，边框 `border-[#3d9a9a]`
- **NPC 头像**：渐变 `from-[#c9a227] to-[#a08020]`，边框 `border-[#f0d878]`

## 动效规范

### 文字打印效果
- 日志内容逐字显示，模拟古籍书写感
- 打印速度：30ms/字
- 配合轻微打字音效

### 呼吸光效
- 灵气相关的元素（💎🌸✨）添加微弱呼吸光效
- 周期：3秒
- 透明度变化：0.7 → 1.0 → 0.7

### 悬停反馈
- 按钮悬停时边框高亮
- 列表项悬停时背景色微变
- 使用 CSS transition，过渡时间 200ms

### 浮动动画
- 装饰元素上下浮动，周期 3s

### 金色光晕
- 重要头像/图标添加金色阴影脉动
- `rgba(201, 162, 39, 0.3)` → `rgba(201, 162, 39, 0.6)`

### 自定义滚动条
- 滑块色：`#2d5a5a`，悬停：`#3d7a7a`
- 轨道色：`transparent`

## 设计原则总结

- **深色优先**：所有页面以深色为底，保护眼睛，营造神秘氛围
- **文字为主**：以文字叙述为核心，ASCII 艺术为辅助装饰
- **符号表意**：使用 Emoji 和 ASCII 符号快速传达信息
- **层次分明**：通过边框、缩进、颜色区分信息层级
- **古风古韵**：语言风格、用词选择贴合修仙世界观
- **克制留白**：适当留白，避免信息过载，保持界面清爽
- **青色为骨**：以深青色系为界面骨架，呼应灵气/修仙意境
- **金色为魂**：以道韵金为强调色，象征仙缘/道韵/传承
