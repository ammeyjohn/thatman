## 设计概述

### 设计理念
以"上古劫后遗落世道"为核心，营造深邃、神秘的修仙氛围。界面采用深色系为主，配合古朴的文字排版，让玩家仿佛置身于残破的修仙典籍之中。

### 核心氛围词
- **古朴**：如上古残卷，承载岁月痕迹
- **深邃**：如青墟之夜，蕴含无尽可能
- **静谧**：如闭关洞府，修心之所
- **玄妙**：如天地法则，因果循环

## 动效建议

### 文字打印效果
- 日志内容逐字显示，模拟古籍书写感
- 打印速度：30ms/字
- 配合轻微打字音效

### 呼吸光效
- 灵气相关的元素（💎🌸✨）添加微弱呼吸光效
- 周期：3秒
- 透明度变化：0.7 → 1.0 → 0.7

### 悬停反馈
- 按钮悬停时边框高亮
- 列表项悬停时背景色微变
- 使用 CSS transition，过渡时间 200ms

## 设计原则总结

- **深色优先**：所有页面以深色为底，保护眼睛，营造神秘氛围
- **文字为主**：以文字叙述为核心，ASCII 艺术为辅助装饰
- **符号表意**：使用 Emoji 和 ASCII 符号快速传达信息
- **层次分明**：通过边框、缩进、颜色区分信息层级
- **古风古韵**：语言风格、用词选择贴合修仙世界观
- **克制留白**：适当留白，避免信息过载，保持界面清爽
