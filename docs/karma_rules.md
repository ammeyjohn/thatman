# 因果业力规则

## 1. 核心法则

- **相遇即有因果，恩怨皆有回响**：玩家与NPC/实体的每次重要交互都可能产生因果
- **修心重于修为**：业力影响道心稳固、突破成功率、NPC态度
- **善恶有报**：善行积功德，恶行结业障，因果循环不可违逆

## 2. 因果类型与业力值

| 因果类型 | 标识 | 善行业力值 | 恶行业力值 | 触发场景 |
|----------|------|-----------|-----------|---------|
| 恩情 | `grace` | +10~+50 | — | 救助、赠宝、传法、助人 |
| 仇怨 | `enmity` | — | -50~-10 | 杀戮、掠夺、羞辱、背叛 |
| 同门情谊 | `fellowship` | +5~+15 | -5~-15 | 同门互助/同门相残 |
| 知己之交 | `friendship` | +10~+30 | -20~-40 | 知己论道/背叛知己 |
| 契约关系 | `contract` | +5~+20 | -10~-30 | 履约/违约 |
| 陌路相逢 | `neutral` | 0 | 0 | 无因果交集 |

## 3. 善恶等级与效果

| 等级 | 业力范围 | 称号 | 效果 |
|------|---------|------|------|
| 5 | ≥200 | 功德圆满 | 突破成功率+20%，NPC好感提升，偶得机缘 |
| 4 | 100~199 | 善行卓著 | 突破成功率+10%，NPC好感略升 |
| 3 | -99~99 | 因果清净 | 无特殊效果 |
| 2 | -199~-100 | 业障缠身 | 突破成功率-10%，心魔概率增加 |
| 1 | ≤-200 | 罪孽深重 | 突破成功率-20%，天劫概率增加，NPC敌意提升 |

## 4. 因果记录触发时机

以下场景**必须**调用 `record_karma` 记录因果：
- 救助受伤/危难中的NPC → 恩情因果
- 击杀/伤害无辜NPC → 仇怨因果
- 掠夺他人财物、资源 → 仇怨因果
- 赠予他人宝物、功法 → 恩情因果
- 与他人缔结契约 → 契约因果
- 违反已缔结的契约 → 契约因果（恶）
- 同门之间重大互助/伤害 → 同门因果
- 知己之间深度交流/背叛 → 知己因果

以下场景**不需要**记录因果：
- 日常闲聊、询问路线
- 查看状态、装备、背包
- 普通交易（等价交换）
- 修炼、赶路、采集等自身行为

## 5. 因果了结规则

当玩家了结一段因果时，调用 `resolve_karma` 工具：
- 恩情：报恩(+15) / 忘恩负义(-30)
- 仇怨：复仇了结(+10) / 宽恕(+25)
- 同门：分别(0) / 重聚(+5)
- 知己：背叛(-35) / 加深(+10)
- 契约：履约(+10) / 违约(-20)

## 6. 业力对修为的影响

- GM 在判定境界突破时，**必须**考虑玩家当前善恶等级
- 功德圆满/善行卓著：突破时可在 dialog 中描述"道心通明，天地感应"等正面效果
- 业障缠身/罪孽深重：突破时需在 dialog 中描述"心魔侵扰"、"道心不稳"等负面效果，增加突破失败概率
- 因果未了结时，相关因果会在突破时产生干扰（如仇人寻仇、恩人求助）

## 7. player_update 业力字段

当业力发生变化时，**必须**在 `player_update` 中包含以下字段：
- `karma`：当前业力值（整数）
- `karma_level`：善恶等级（1-5）
- `karma_title`：善恶称号（如"因果清净"）
