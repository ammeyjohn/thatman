# 功法配置
制定各类功法品质与属性规则，明确习得及升级条件，为AI生成功法、更新宗门功法库提供依据。

## 功法品级
| 品级 | 对应境界 | 获取难度 | 传承状态 |
|------|---------|---------|---------|

## 功法习得条件
| 条件 | 说明 |
|------|------|
| 境界要求 | 功法品级对应最低境界 |
| 灵根匹配 | 部分功法需对应属性灵根（如雷属性功法需雷灵根） |
| 悟性要求 | 品级越高，悟性要求越高 |
| 传承条件 | 宗门功法需对应弟子等级，散修功法需机缘获取 |

## 功法升级规则
| 升级方式 | 说明 |
|---------|------|
| 修炼领悟 | 反复修炼提升熟练度，解锁功法深层效果 |
| 论道交流 | 与他人论道碰撞，弥补悟道盲区，加速领悟 |
| 秘境感悟 | 在上古秘境感悟道韵，补全残缺功法 |
| 古法拼凑 | 收集残缺典籍，拼凑完整古法 |

## 残缺功法规则
| 规则 | 说明 |
|------|------|
| 残缺程度 | 仙阶及以上功法多数残缺，分为1/4、1/2、3/4残缺 |
| 补全方式 | 秘境寻宝、论道交流、古法拼凑 |
| 残缺影响 | 仅能修炼已补全部分，残缺部分修炼有走火入魔风险 |
| 补全加成 | 每补全1/4，功法效果+10% |
