# 境界配置
定义完整修士境界体系，制定突破、属性、权限相关规则，是角色修炼、战力计算的底层数据基础。

## 小段位划分规则
每个大境界细分为5个小段位：初期 → 中期 → 后期 → 巅峰 → 圆满

## 境界层级总览
青墟大世界修行境界分为阶段：人界境界、飞升灵界境界、飞升仙界境界。

## 突破规则
### 通用突破条件
| 条件类型 | 说明 | 影响程度 |
|---------|------|---------|
| 灵力积累 | 当前境界灵力达到上限 | 必须满足 |
| 心境修为 | 道心稳固程度达标 | 必须满足 |
| 资质要求 | 灵根品质影响突破成功率 | 影响成功率 |
| 悟性要求 | 对功法的领悟程度 | 影响成功率 |
| 外部条件 | 灵气浓度、突破丹药、护法等 | 辅助提升成功率 |

| 目标境界 | 特殊要求 | 失败惩罚 |
|---------|---------|---------|
| 筑基期 | 需筑基丹辅助，完成筑基周天 | 修为跌落，根基受损 |
| 金丹期 | 需渡过丹劫（小天劫） | 金丹碎裂，修为尽废 |
| 元婴期 | 需元婴化形，神识蜕变 | 元婴夭折，神识受损 |

| 炼虚期 | 需炼虚合道，感悟虚空 | 虚实反噬，修大为损 |

| 渡劫期 | 需渡过九重天劫（大天劫） | 渡劫失败，身死道消 |

## 境界压制规则
| 压制类型 | 效果 | 适用场景 |
|---------|------|---------|
| 大境界压制 | 高境界对低境界全属性+50%压制 | 相差1个大境界以上 |
| 小段位压制 | 同境界高段位对低段位+20%压制 | 同境界不同段位 |
| 神识压制 | 高境界神识可压制低境界神识探查 | 神识范围差距2倍以上 |
| 威压压制 | 高境界释放威压，低境界属性下降 | 主动释放威压 |

## 境界显示规则
| 场景 | 显示规则 | 说明 |
|-----|---------|------|
| 同境界 | 显示完整境界+小段位 | 如：金丹期中期 |
| 低境界看高境界 | 显示"深不可测"或大致范围 | 无法准确感知 |
| 高境界看低境界 | 显示完整境界+小段位 | 一目了然 |
| 特殊功法/法宝 | 可隐藏或伪装境界 | 视具体效果而定 |
