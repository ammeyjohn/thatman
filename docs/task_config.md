# 任务配置
规范任务难度分级、类型及奖励区间，为AI生成各类任务提供标准，平衡任务收益与角色境界。

## 任务难度分级
| 难度 | 标识 | 适用境界 | 推荐战力 | 失败风险 |
|------|------|---------|---------|---------|
| 极易 | 灰色 | 引气入体 | 基础 | 极低 |
| 简单 | 白色 | 炼气期 | 基础×1.5 | 低 |
| 普通 | 绿色 | 筑基期 | 基础×3 | 中等 |
| 困难 | 蓝色 | 金丹期 | 基础×6 | 较高 |
| 精英 | 紫色 | 元婴期 | 基础×12 | 高 |
| 极难 | 橙色 | 化神期 | 基础×25 | 极高 |
| 传说 | 金色 | 炼虚期及以上 | 基础×50+ | 死亡风险 |

## 任务类型

### 主线任务
| 属性 | 说明 |
|------|------|
| 定位 | 推进剧情，引导玩家体验世界观 |
| 特点 | 不可放弃，必须完成才能推进 |
| 奖励 | 经验、功法、关键道具、剧情解锁 |
| 来源 | 剧情自动触发 |

### 支线任务
| 属性 | 说明 |
|------|------|
| 定位 | 丰富世界观，拓展角色关系 |
| 特点 | 可选完成，影响因果与NPC关系 |
| 奖励 | 灵石、丹药、装备、NPC好感 |
| 来源 | NPC对话、场景探索、事件触发 |

### 宗门任务
| 属性 | 说明 |
|------|------|
| 定位 | 宗门弟子日常，提升宗门贡献 |
| 特点 | 需宗门弟子身份，限时完成 |
| 奖励 | 宗门贡献、灵石、宗门功法权限 |
| 来源 | 宗门管事处、长老指派 |

### 奇遇任务
| 属性 | 说明 |
|------|------|
| 定位 | 随机触发的特殊机缘 |
| 特点 | 不可重复，时效性强，风险与收益并存 |
| 奖励 | 稀有道具、上古传承、道韵感悟 |
| 来源 | 秘境探索、特殊天气、灵潮期间 |

### 历练任务
| 属性 | 说明 |
|------|------|
| 定位 | 修士成长历练，打磨道心 |
| 特点 | 可重复，难度随境界提升 |
| 奖励 | 修为、心性提升、天材地宝 |
| 来源 | 宗门指派、自行历练 |

## 任务奖励区间
| 难度 | 灵石 | 经验 | 道具品级 | 额外奖励概率 |
|------|------|------|---------|-------------|
| 极易 | 10~50下品 | 基础×1 | 凡品 | 5% |
| 简单 | 50~200下品 | 基础×2 | 凡品~良品 | 10% |
| 普通 | 1~10中品 | 基础×5 | 良品~上品 | 15% |
| 困难 | 10~50中品 | 基础×10 | 上品 | 20% |
| 精英 | 1~10上品 | 基础×20 | 上品~极品 | 25% |
| 极难 | 10~50上品 | 基础×50 | 极品 | 30% |
| 传说 | 1~10极品 | 基础×100 | 极品~上古品 | 40% |

## 任务因果影响
| 任务行为 | 因果影响 | 影响范围 |
|---------|---------|---------|
| 救助NPC | 产生恩情因果 | 被救助NPC及关联NPC |
| 击杀NPC | 产生仇怨因果 | 被击杀NPC的师门/亲友 |
| 完成宗门任务 | 提升同门好感 | 宗门内NPC |
| 帮助灵族 | 获得灵族信任 | 灵族NPC |
| 掠夺资源 | 产生仇怨因果 | 被掠夺方 |
| 违约行为 | 契约反噬，道心受损 | 自身 |

## 任务刷新规则
| 任务类型 | 刷新周期 | 上限 |
|---------|---------|------|
| 宗门日常 | 每日 | 5个 |
| 宗门周常 | 每周 | 3个 |
| 历练任务 | 每日 | 3个 |
| 奇遇任务 | 随机触发 | 无上限 |
| 支线任务 | 随剧情推进 | 无上限 |
