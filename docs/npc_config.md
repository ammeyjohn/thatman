# NPC配置
划分NPC类型与行为逻辑，规范AI生成NPC的各项参数，适配交易、奇遇、战斗等全部交互场景。

## 生灵大类
| 类型 | 说明 | 可选阵营 | 特征 |
|------|------|---------|------|
| 人族修士 | 主流生灵，数量最多 | 全阵营 | 修行体系完善，世界演化核心 |
| 草木精怪 | 灵草古树化形 | 灵族/中立 | 性情温和，擅长感知灵气、炼制草药 |
| 妖兽灵兽 | 野兽吸收灵气演化 | 灵族/中立 | 肉身强悍，高阶可化形，部分可缔结契约 |
| 上古遗族 | 上古大劫存活者 | 中立 | 数量稀少，掌握上古古法，隐居秘境 |

## 资质体系
### 灵根品级
| 品级 | 属性 | 灵气吸收 | 悟道加成 | 稀有度 |
|------|------|---------|---------|--------|
| 先天灵根 | 单一属性 | 极快 | +30% | 极罕见 |
| 变异灵根 | 双属性融合 | 较快 | +20% | 稀少 |
| 杂灵根 | 三属性及以上 | 中等 | 无加成 | 常见 |
| 伪灵根 | 驳杂 | 极慢 | -20% | 较常见 |

### 辅助天赋
| 天赋 | 影响 | 高天赋表现 |
|------|------|-----------|
| 悟性 | 悟道速度、功法领悟 | 快速掌握残缺古法，举一反三 |
| 心性 | 道心稳固程度 | 抵御心魔、执念、因果干扰 |
| 灵觉 | 感知灵气/机缘/危险 | 快速找到灵脉、天材地宝 |

## 因果关系与行为逻辑
| 因果类型 | 态度 | 行为倾向 |
|---------|------|---------|
| 恩情 | 恭敬、感激、亲近 | 主动相助、分享机缘、不与施恩方为敌 |
| 仇怨 | 冰冷、敌视、警惕 | 主动攻击/嘲讽、抢夺机缘、追寻复仇 |
| 同门 | 亲切、尊重 | 相互助助、论道历练、遵守门规 |
| 知己 | 坦诚、信任 | 共赴险境、共享资源、相互指点 |
| 陌路 | 淡然、冷漠 | 擦肩而过，有冲突优先自保 |
| 契约 | 严谨、守信 | 严格履约，违约受反噬 |

## 记忆机制
| 类型 | 留存时长 | 内容 | 影响 |
|------|---------|------|------|
| 短时记忆 | 7天 | 临时交集（短暂对话、偶遇） | 不影响长期行为 |
| 长时记忆 | 永久 | 深远因果（救命、血仇、传承、知己） | 永久影响行为 |

### 记忆触发
| 触发方式 | 说明 |
|---------|------|
| 重逢触发 | 再次相遇自动调取长时记忆，调整态度行为 |
| 事件触发 | 遇相关事件唤醒记忆（如见同类资源想起被掠） |
| 言语触发 | 交谈提及过往因果时唤醒记忆，调整对话态度 |

## NPC生成参数
| 参数 | 取值范围 | 说明 |
|------|---------|------|
| 生灵类型 | 人族/草木精怪/妖兽灵兽/上古遗族 | 决定基础属性与成长方式 |
| 心境类型 | 8种心境 | 终身不变，决定言行与行为逻辑 |
| 阵营归属 | 正道/邪道/中立/灵族 | 决定初始立场与关系 |
| 身份标签 | 见身份标签体系 | 决定社会地位与权限 |
| 灵根品级 | 先天/变异/杂灵根/伪灵根 | 决定修行起点 |
| 辅助天赋 | 悟性/心性/灵觉 | 决定修行上限 |
| 初始境界 | 引气~大乘 | Demo仅开放引气~大乘 |
| 因果列表 | 动态生成 | 记录与玩家/其他NPC的恩怨关系 |
