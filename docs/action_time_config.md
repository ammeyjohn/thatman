# 行为耗时配置
统筹各类行为的游戏耗时范围与修正规则，为AI判定行为耗时提供标准依据，平衡游戏节奏与真实感。

## 即时行为（0~30分钟）
| 行为类型 | 基础耗时 | 影响因素 |
|---------|---------|---------|
| 日常对话 | 5~15分钟 | 无 |
| 查看物品/装备 | 5~10分钟 | 无 |
| 服用丹药 | 10~30分钟 | 丹药品级 |
| 简单探索 | 15~30分钟 | 区域危险度 |
| 整理行囊 | 5~10分钟 | 物品数量 |
| 休息进食 | 15~30分钟 | 无 |

## 短时行为（30分钟~2时辰）
| 行为类型 | 基础耗时 | 影响因素 |
|---------|---------|---------|
| 区域内移动 | 30~60分钟 | 区域大小 |
| 采集资源 | 30~90分钟 | 资源稀有度 |
| 基础打坐调息 | 1~2时辰 | 境界、灵气浓度 |
| 简单交易 | 30~60分钟 | 无 |
| 寻找NPC | 30~60分钟 | 区域大小、NPC位置 |
| 简单炼丹 | 1~2时辰 | 丹药品级、熟练度 |

## 中时行为（2~6时辰）
| 行为类型 | 基础耗时 | 影响因素 |
|---------|---------|---------|
| 跨区域赶路 | 2~4时辰 | 距离、地形 |
| 深度修炼 | 2~4时辰 | 功法等级、灵气浓度 |
| 炼丹/炼器 | 2~6时辰 | 品级、熟练度 |
| 探索秘境 | 2~6时辰 | 秘境规模、危险度 |
| 宗门任务 | 2~4时辰 | 任务难度 |

## 长时行为（6时辰~数天）
| 行为类型 | 基础耗时 | 影响因素 |
|---------|---------|---------|
| 闭关苦修 | 1~3天 | 境界、资源 |
| 境界突破 | 6时辰~2天 | 境界差距、辅助资源 |
| 远程赶路 | 1~5天 | 距离、交通工具 |
| 大型炼制 | 1~3天 | 品级、材料 |

## 耗时修正规则
| 修正因素 | 效果 | 适用行为 |
|---------|------|---------|
| 灵气浓度高 | 修炼耗时-20%~50% | 修炼、突破 |
| 聚灵阵/灵脉 | 修炼耗时-30%~60% | 修炼、突破 |
| 辅助丹药 | 突破耗时-20%~40% | 突破 |
| 高阶功法 | 修炼耗时-10%~30% | 修炼 |
| 境界高于行为难度 | 耗时-20%~50% | 通用 |
| 境界低于行为难度 | 耗时+20%~100% | 通用 |
| 恶劣天气 | 赶路耗时+20%~50% | 赶路、探索 |
| 法宝辅助 | 耗时-10%~30% | 炼丹、炼器、探索 |
| 修炼场地品质 | 修炼耗时-10%~40% | 修炼 |

## time_cost 字段规则
- 单位：游戏分钟
- 即时行为（日常对话、查看状态等）：设为 0
- 耗时行为：必须查询本文档确定基础范围，在范围内根据上下文调整
- 修正规则叠加时，总修正不超过基础耗时的±70%
- 每次回复必须包含此字段，不可省略
