# 游戏说明书
阐述游戏定位、核心特色、运行规则与交互指令，面向玩家，作为游戏入门指导文档。

## 基础规则
- 时间系统：所有行动都会消耗游戏时辰，时辰实时流转；修炼、突破、赶路等行为难度越高，耗时越长，特殊道具能够缩短行动耗时。
- 境界瓶颈：修为积攒至满值后无法自动升级，所有境界进阶都需要配合打坐、优质修炼场地、辅助道具，主动发起突破，且突破存在成败机制。
- 交互指令：内置专属查询指令，指令页面统一采用对称ASCII标题格式，随时可调用查看游戏数据，不影响剧情进度。

## 核心玩法

### 剧情抉择
剧情流程随玩家选择推进，每一个关键节点提供多项行动选项。不同选择会改变剧情走向、解锁隐藏机缘、影响宗门好感，同时解锁全新场景与后续玩法分支。

### 境界修炼
修炼是游戏核心，玩家可自由选择多种修炼方式：
- 基础修炼：打坐、调息、修炼武学功法，稳定积攒修为，夯实修行根基；
- 深度苦修：长时间闭关打坐，额外提升体质属性，降低后续突破失败代价；
- 境界突破：修为满值后，依托灵地、灵液等资源，主动冲击境界瓶颈，提升综合战力与宗门权限。

###  道具资源
- 聚灵类宝物：可在灵气充足区域自动汇聚天地灵气，生成精纯灵液；手持宝物打坐，直接提升整体修炼效率；
- 灵液资源：通用高阶修炼资源，既可日常服用加速修为增长，也能在突破瓶颈时使用，大幅提升突破成功率；
- 功能道具：包含身份凭证、服饰、丹药等，用于解锁场景权限、恢复状态、辅助修行与实战。

### 场景探索
玩家可自由穿梭各个地图场景，包含村落、山林、宗门各大功能区域。宗门内拥有修炼场、丹药阁、居所、管事处等专属区域，分别提供优质修炼环境、资源兑换、休息存档、任务登记等服务，助力玩家快速成长。

### 宗门成长
以宗门弟子身份完成登记、日常任务、宗门考核，提升自身弟子等级。解锁高阶修炼场地、稀缺丹药、独家宗门功法，从外门弟子逐步晋升，解锁更多高阶修仙内容。

## 游戏指令
- /状态：查看角色基础属性、境界、体质、战力、当前游戏时间与身份。
- /功法: 查看当前已习得功法。
- /装备：查看当前装备等。
- /储物：查看当前储物物品等。
- /事件：查看全部待办主线、支线任务，明确现阶段游戏目标。
- /历史：查阅过往所有剧情、关键行动与成长经历。
- /全部：一站式查询状态、装备、储物物品、习得功法、待办事件所有内容。
- /帮助：查看所有指令与说明。

## 文件系统
- game_manual.md
阐述游戏定位、核心特色、运行规则与交互指令，面向玩家，作为游戏入门指导文档。
- world_config.md
统筹地图、天气两类环境数据，设定环境增益与负面效果，影响修炼、资源、奇遇，辅助AI生成动态场景。
- task_config.md
规范任务难度分级、类型及奖励区间，为AI生成各类任务提供标准，平衡任务收益与角色境界。
- level_config.md
定义完整修士境界体系，制定突破、属性、权限相关规则，是角色修炼、战力计算的底层数据基础。
- item_config.md
统一管理全品类道具资源，规范道具属性、获取方式与使用效果，支撑AI掉落、兑换、奖励等功能。
- skill_config.md
制定各类功法品质与属性规则，明确习得及升级条件，为AI生成功法、更新宗门功法库提供依据。
- npc_config.md
划分NPC类型与行为逻辑，规范AI生成NPC的各项参数，适配交易、奇遇、战斗等全部交互场景。
- guild_config.md
设定宗门势力分级标准，管控宗门资源、弟子体系与势力关系，辅助AI生成宗门及处理宗门相关玩法。
